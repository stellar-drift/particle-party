# particle party

simple javascript site built for a class. showcases event handling, objects, DOM manipulation, button logic, form validation, radio selection, use of p5.js, and more.



## demo:

https://stellar-drift.github.io/particle-party/



## how to?

enter your name and press the button!  
you can choose from five canvas colors :)  
reset if desired.  



## built with:

- html
- css
- javascript
